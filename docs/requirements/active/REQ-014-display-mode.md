# REQ-014: 添加键盘数字键切换显示模式

**优先级**: P2
**类型**: 新功能
**预估工时**: 20 分钟
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望按数字键 1-4 只显示对应的几何体，
以便单独观察每种几何体的效果。

## 验收标准

- [ ] 按 1 只显示立方体
- [ ] 按 2 只显示球体
- [ ] 按 3 只显示圆柱体
- [ ] 按 4 只显示圆锥体
- [ ] 按 0 显示全部
- [ ] 按 5 显示线框模式
