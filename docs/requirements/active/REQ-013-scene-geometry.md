# REQ-013: 在场景中展示多种几何体

**优先级**: P1
**类型**: 新功能
**预估工时**: 30 分钟
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望在场景中看到立方体、球体、圆柱体等几何体，
以便理解不同几何体的渲染效果。

## 验收标准

- [ ] 场景中展示立方体、球体、圆柱体、圆锥体
- [ ] 每种几何体有不同位置
- [ ] 可通过键盘切换几何体
