# REQ-012: 添加更多几何体（球体、圆柱体）

**优先级**: P1
**类型**: 新功能
**预估工时**: 1 小时
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望能够渲染球体和圆柱体，
以便创建更丰富的 3D 场景。

## 验收标准

- [ ] MeshUtils 提供 createSphere() 方法
- [ ] MeshUtils 提供 createCylinder() 方法
- [ ] 球体顶点数据正确（平滑法线）
- [ ] 圆柱体顶点数据正确
- [ ] 添加单元测试

## 技术要点

- 球体使用 UV 球算法（经纬度细分）
- 圆柱体使用侧面 + 顶底盖
- 法线从中心向外计算

## 测试要求

- 单元测试验证顶点数量
- 验证法线方向
