# REQ-001: 改进纹理立方体渲染效果

**优先级**: P1
**类型**: 新功能
**创建日期**: 2026-02-21
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望看到一个正确渲染的纹理立方体（6个面都有正确的纹理贴图），
以便理解 OpenGL 纹理映射的工作原理。

## 背景

用户反馈当前渲染结果"有点诡异"，可能的问题：
1. 纹理坐标可能不正确
2. 摄像机视角或距离可能不合适
3. 光照计算可能影响纹理显示

## 验收标准

- [ ] 立方体居中显示，完整可见
- [ ] 6个面都正确显示纹理（container2.png）
- [ ] 纹理方向正确（没有旋转或镜像）
- [ ] 缓慢旋转动画，可以看清每个面
- [ ] 渲染效果清晰，无视觉异常

## 技术建议

1. 检查纹理坐标是否正确
2. 调整摄像机距离，确保立方体完整可见
3. 简化着色器，先确保纹理正确显示
4. 使用更慢的旋转速度（0.3x）
5. 截图验证每个面的纹理
