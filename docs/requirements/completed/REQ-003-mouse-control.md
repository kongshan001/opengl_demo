# REQ-003: 添加鼠标视角控制

**优先级**: P1
**类型**: 新功能
**创建日期**: 2026-02-21
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望用鼠标移动来旋转视角，
以便更自由地观察场景。

## 验收标准

- [ ] 鼠标移动控制视角旋转
- [ ] 鼠标滚轮控制缩放（FOV）
- [ ] 视角旋转平滑
- [ ] 俯仰角有上下限（避免翻转）

## 技术建议

使用 GLFW 的鼠标回调，Camera 类已有 processMouseMovement 和 processMouseScroll。
