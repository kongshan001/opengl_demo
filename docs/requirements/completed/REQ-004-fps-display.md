# REQ-004: 添加FPS显示

**优先级**: P2
**类型**: 新功能
**创建日期**: 2026-02-21
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望在窗口标题栏看到FPS和帧时间，
以便监控渲染性能。

## 验收标准

- [ ] 窗口标题显示FPS
- [ ] 显示帧时间(ms)
- [ ] 每秒更新一次（不每帧更新）

## 技术建议

在 run() 循环中计算 FPS，用 glfwSetWindowTitle 更新标题。
