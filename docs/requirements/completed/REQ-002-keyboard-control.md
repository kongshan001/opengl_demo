# REQ-002: 添加键盘交互控制

**优先级**: P1
**类型**: 新功能
**创建日期**: 2026-02-21
**状态**: 开发中

## 用户故事

作为一个 OpenGL 学习者，
我希望用键盘 WASD 控制摄像机移动，
以便从不同角度观察场景。

## 验收标准

- [ ] W/S 控制前后移动
- [ ] A/D 控制左右移动
- [ ] Q/E 控制上下移动
- [ ] 移动速度合理（不过快/过慢）
- [ ] ESC 退出程序

## 技术建议

已有 Camera 类的 processKeyboard 方法，只需确保正确调用。
